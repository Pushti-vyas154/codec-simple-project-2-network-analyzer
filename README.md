# Network Traffic Analysis using Wireshark

## 📌 Objective

Capture and analyze network traffic to understand communication protocols and identify network behavior.

## 🛠 Tools Used

* Wireshark

## 🔍 Analysis Performed

* DNS Analysis (Domain resolution)
* TCP Analysis (3-way handshake & communication)
* TLS Analysis (Encrypted traffic inspection)

## 📊 Key Findings

* DNS resolves domain names into IP addresses
* Majority of traffic uses HTTPS (port 443)
* TLS ensures secure encrypted communication
* Communication with cloud/CDN servers observed
* No malicious activity detected

* `report.pdf` → Detailed analysis report
* `screenshots/` → Packet analysis images
